# AstroMatch — Kurulum Rehberi

## 1. Terminal'de proje klasörüne gir
```
cd astromatch
```

## 2. Bağımlılıkları kur (bir kez çalıştır)
```
npm install
```

## 3. Uygulamayı başlat
```
npm run dev
```

Tarayıcıda http://localhost:5173 adresine git — uygulama açılacak!

## 4. .env.local dosyasını düzenle
Anthropic API key için: https://console.anthropic.com
